Installation

De la carte SD au premier écran,
pas à pas.

Une vingtaine de minutes, entièrement à distance - ni écran ni clavier sur le Pi.

Avant de commencer

Il vous faut :

  • le montage câblé (voir la notice de câblage) - pas indispensable pour installer, mais utile pour voir l'écran s'allumer à la fin ;
  • une carte micro SD et un PC sous Windows (la procédure ci-dessous utilise PowerShell, intégré à Windows) ;
  • une clé API Mistral ou Claude (gratuite à créer ; on la renseigne tout à la fin).

L'installation, étape par étape

  1. Préparez la carte SD. Avec Raspberry Pi Imager, flashez Raspberry Pi OS Lite :

    • Pi Zero 2 W / Pi 3 : version 64-bit ;
    • Pi Zero W (v1) : version 32-bit - c'est la seule proposée pour ce modèle (processeur ARMv6), c'est normal.

    Dans les réglages (⚙) de l'imageur : activez SSH, créez l'utilisateur minitel avec un mot de passe, et renseignez votre WiFi. Le Pi fonctionnera ensuite tout seul, sans écran ni clavier.

  2. Connectez-vous au Pi en SSH. Insérez la carte, branchez le Pi, attendez une minute qu'il rejoigne le WiFi. Sur votre PC, ouvrez PowerShell et trouvez l'adresse IP du Pi (le filtre cible les adresses des Raspberry) :

    arp -a | Select-String "b8-27-eb|dc-a6-32|e4-5f-01|d8-3a-dd|2c-cf-67|28-cd-c1"

    Connectez-vous en remplaçant l'IP par celle obtenue :

    ssh minitel@192.168.1.42

    Saisissez le mot de passe défini à l'étape 1. C'est normal qu'aucun caractère ne s'affiche pendant la frappe : c'est une sécurité.

    Astuce : dans PowerShell, un clic droit colle le presse-papiers.

  3. Mettez le système à jour et installez git (qui servira ensuite aux mises à jour en un clic) :

    sudo apt update
sudo apt install git

  4. Téléchargez puis installez MINITEL GPT :

    cd /home/minitel
git clone https://github.com/jherard-fr/minitel-gpt.git
cd minitel-gpt
sudo bash install.sh

    Puis démarrez les services :

    sudo systemctl restart minitel-chatgpt admin-ui wifi-manager

    Voilà, c'est installé ! L'écran d'accueil s'affiche sur le Minitel (sur un Minitel 2, faites Fnct + Sommaire à chaque allumage).

  5. Renseignez la clé API. Rien à éditer à la main : ouvrez l'interface d'administration (http://<ip-du-pi>:8080, mot de passe mistral ; l'IP s'affiche aussi sur le Minitel via la touche Guide). Dans l'onglet Paramètres, choisissez le fournisseur d'IA et collez la clé :

    • Clé Mistral : admin.mistral.ai
    • Clé Claude : platform.claude.com

    Pour chaque fournisseur, un menu propose les modèles disponibles avec leur coût et leur pertinence.

Et voilà !

✅ Votre Minitel dialogue avec l'IA

Tapez une question sur le clavier du Minitel, validez avec Envoi : la réponse s'affiche à l'écran. Tout le reste - personnalités, textes d'accueil, fichiers de connaissance, mises à jour - se règle depuis l'interface d'administration, sans jamais reprendre la ligne de commande.

Le dossier installé est un clone git : les mises à jour se font en un clic depuis l'admin (onglet Paramètres). Le détail complet et le code source sont sur GitHub.
