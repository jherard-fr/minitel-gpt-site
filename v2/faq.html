FAQ

Les questions qu'on nous pose (souvent au clavier)

Minitel, Raspberry Pi et intelligence artificielle : les réponses aux questions les plus posées.

01 · Câblage Minitel ↔ Raspberry Pi

Comment connecter un Minitel à un Raspberry Pi ? +

On relie le Minitel au Raspberry Pi via sa prise péri-informatique (DIN 5 broches) et un adaptateur USB-série FTDI FT232RL (jumper sur 5V). Trois fils suffisent :

  • plot 1 (Rx du Minitel) → TXD du FTDI ;
  • plot 3 (Tx du Minitel) → RXD du FTDI ;
  • plot 2 (masse) → masse du FTDI.

Les plots 4 et 5 ne doivent pas être utilisés. Le FTDI se branche ensuite en USB sur le Pi (port /dev/ttyUSB0), liaison série à 1200 bauds, 7E1. Détail complet et schéma sur la notice de câblage.

Le montage fonctionne-t-il avec un Minitel 1 et un Minitel 2 ? +

Oui, les deux fonctionnent, avec le même câblage. Sur un Minitel 1 (Telic/Alcatel), le mode péri-informatique est actif par défaut. Sur un Minitel 2, il faut faire Fnct + Sommaire à chaque allumage pour sortir de l'écran Répertoire et passer en Vidéotex natif ; ensuite le comportement est identique aux deux modèles.

Quel Raspberry Pi choisir ? +

Un Raspberry Pi Zero W suffit largement (le moins cher, il tient dans un petit boîtier). Le Pi Zero 2 W et le Pi 3 fonctionnent aussi. Sur le Zero W (v1, ARMv6), flasher la version 32-bit de Raspberry Pi OS Lite ; sur Zero 2 W et Pi 3, la version 64-bit.

Le Raspberry Pi a-t-il besoin d'un écran ou d'un clavier ? +

Non. Le WiFi se configure au flashage de la carte SD (ou via le hotspot de secours créé automatiquement par le Pi si le réseau est absent). Toute la configuration se fait ensuite à distance : SSH pour l'installation, puis interface web d'administration pour l'IA et les personnalités. Le Minitel devient le seul écran et clavier utiles au quotidien.

02 · Connexion à une IA (Mistral, Claude)

Comment connecter le Raspberry Pi à une intelligence artificielle (LLM) ? +

Le Raspberry Pi appelle l'API d'un fournisseur d'IA en HTTPS via une clé API : Mistral (modèle mistral-small-latest) ou Claude (Anthropic), au choix. La clé se renseigne sans ligne de commande, depuis l'interface d'administration web du Pi (port 8080, onglet Paramètres). Chaque question tapée au clavier du Minitel part vers ce LLM, et la réponse revient s'afficher à l'écran en Vidéotex, paginée avec la touche Suite.

Faut-il savoir coder pour brancher un Minitel sur un LLM ? +

Non. Le script install.sh configure tout automatiquement sur le Raspberry Pi. Ensuite, l'interface web d'administration permet de choisir le fournisseur d'IA, coller la clé API, créer des « personnalités » de conversation et ajouter des fichiers de connaissance, sans jamais toucher au code après l'installation.

Combien coûte l'utilisation de l'IA sur ce montage ? +

Quelques centimes par soirée d'utilisation avec Mistral ou Claude — les modèles utilisés sont légers et le volume de texte échangé (questions/réponses affichées sur un écran Minitel 40 colonnes) reste faible.

03 · Le projet

Combien coûte un projet Minitel GPT et combien de temps prend le montage ? +

Le matériel (Raspberry Pi Zero W, adaptateur FTDI, câble OTG, carte micro SD) coûte quelques dizaines d'euros. Le montage physique et l'installation logicielle prennent environ une heure quand on dispose déjà du matériel — voir le guide d'installation pas à pas.

Le code de MINITEL GPT est-il open source ? +

Oui, l'intégralité du code (terminal Minitel, gestion WiFi autonome, interface d'administration web) est publiée en open source sur GitHub, librement consultable, installable et modifiable. Voir la page GitHub.
