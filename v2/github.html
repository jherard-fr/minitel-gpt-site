Code GitHub

Un projet 100 % open source

Tout le code de MINITEL GPT est proposé librement et en open source sur GitHub.

Vous êtes libre de le consulter, l'installer sur votre propre Minitel, le modifier et le réutiliser. Vous y trouverez l'intégralité du projet : le terminal qui dialogue avec l'IA, l'interface web d'administration, la gestion autonome du WiFi, ainsi que la procédure d'installation complète.

Une question, une amélioration, une contribution ? Tout passe par le dépôt GitHub.
